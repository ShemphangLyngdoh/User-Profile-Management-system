# User Profile Management System

## 📌 Project Description
This is a simple web-based User Profile Management System developed using HTML, CSS, and JavaScript.  
It allows users to register, log in, and manage their profile information.

---

## 🚀 Features
- User Registration
- User Login Authentication
- Dashboard Display
- Update Profile
- Delete Account
- Data stored using Local Storage

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📂 How It Works
1. User registers with name, email, and password  
2. Data is stored in browser localStorage  
3. User logs in using credentials  
4. Dashboard displays user information  
5. User can update or delete their account
